Маски очистки кеша:

  • Если параметр пустой, то очищается весь кеш
  • Можно указать одну маску /img/* Из кеша будут удалены все файлы, начинающиеся на /img/ (/img/one.jpg, /img/two.jpg).
  • Можно указать несколько масок через запятую /img/*, /video/* Из кеша будут удалены все файлы, начинающиеся на /img/ и /video/ (/img/one.jpg, /img/two.jpg, /video/one.mp4, /video/video.avi и т.п.).
  • Если указать маску без *, то из кеша будет удалён только файл с url, строго совпадающий с маской. К примеру, masks=/img/my_super_image.jpg?id=8193737182253 Из кеша будет удалён файл, url которого строго совпадает с /img/my_super_image.jpg?id=8193737182253.
